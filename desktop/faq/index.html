Popular questions
Some of the more common questions we get.

Is this 100% free accounting software?

Yes. You can use the software as long as you like, use all the features and enter as much data as required.

If you give away your software for free, how do you make money?

Desktop edition is single-user software. Businesses requiring multi-user capabilitities or remote access will typically use cloud edition or server edition which are paid products.

What is the difference between desktop and cloud edition?

Desktop edition doesn't support multi-user access. If you need multiple users to access your accounting file at the same time, you need cloud edition (or server edition).

If I'm working on Mac, can I send my file to an accountant who is on Windows?

Yes. Manager data is cross-compatible across all operating systems.

If I want to upgrade to cloud edition in future, can I import my data?

Yes. Manager data is cross-compatible across all editions.

Do I have to pay for upgrades?

No. Upgrades are free and always will be free.

What happens to my data when I upgrade?

When you upgrade to the latest version, your data will be automatically rolled-over. Still, we strongly recommend you make regular backups of your data whether you are upgrading or not.
