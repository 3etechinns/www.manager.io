Digital Ocean

Get free $10 credit when you sign up for Digital Ocean. That's enough to run Manager on Digital Ocean servers for 2 months.

Create Digital Ocean account to get free $10 credit

Sign up for Digital Ocean account and go through verification process.

Install Manager

Once you are in control panel, click on Create button in top-right corner, then Droplets.

First select an image and make sure you choose Ubuntu 16.04.3 x64

Then select size of your server. You can go for the cheapest one which is $5 per month.

Digital Ocean is running data centers around the world. Select the location which is the closest to you.

In additional options, select User data option and paste the following two lines into textbox below.

#!/bin/bash
curl -fsSL https://www.manager.io/server/install.txt | bash

Selecting Backups option is also highly recommended.

Then click Create button.

Waiting for server to be created

You will see progress bar indicating the server is being created.

Wait until the process finishes so you can see IP address of your new server.

Your server is running but it will take another 5-10 minutes for Manager to be installed on it. Be patient.

Connecting to the Server

After you wait for 5-10 minutes, note the IP address of your new server.

Open your web-browser and enter xx.xx.xx.xx:8080. where xx.xx.xx.xx is IP address of your server. So for example, if your IP address is 128.199.162.253 then you should enter to your web-browser 128.199.162.253:8080

You should see login screen to Manager Server.

If this is what you see. Congratulations, you are up & running. Just click Login button as default password is empty.

It is highly recommended that you don't connect to your server directly via IP address for security reasons. You should have a domain name pointing to IP address of your server.

For example, if you own domain name mycompany.com, you should create sub-domain accounting.mycompany.com or similar and have it pointed to IP address of your server. If you are not sure how to do this, reach out to your domain registrar or local IT person.

Your server will automatically request free SSL certificate for the domain you choose to use and your connection to your server will be automatically encrypted using HTTPS protocol.
