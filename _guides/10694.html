--- layout: guide title: Reset administrator password tags: users ---

If you are using server edition and forget administrator password, you will have difficulties to log in.

In order to regain access, you will need to reset administrator password.

The password is saved in application data folder where Manager stores all data. In order to reset the password, go to application data folder.

Find the file named password and delete it. This will remove password from administrator account.

You can go back to login screen and enter username administrator and click Login button (do not enter anything into password field).

It is highly recommended, once you log in, you set new password under Users tab so nobody will be able to gain access to your server as administrator without entering any password.
