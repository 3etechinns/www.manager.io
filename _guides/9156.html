By default, Manager will open and operate in English. If you prefer to work with Manager in a different language, you can easily switch.

No matter which language you currently have selected, click the Preferences tab at the top of the Manager window; then click the language link:

=>

Choose your preferred language in the dropdown box that appears:

ClickUpdate. You should see the program in your selected language.

Example
Northwind Traders moves to the Netherlands and decides to switch languages to Dutch. It follows the procedures described above and now sees information in the new language:


Notes
Language is a global preference affecting all businesses you have created in Manager.

The translation process is a collaborative, community effort. See translation progress for more information. If you find incomplete or incorrect translations, you may like to join our translation team. You can suggest new languages at the same link.
