Batch operations streamline large-scale creation, updating, and deletion of information in Manager. They can be used in certain tabs where control accounts contain subsidiary ledgers and bulk editing might be tedious. These tabs are also the ones most useful for migrating information from one business to another. Batch operations function in these tabs:

  • Customers
  • Suppliers
  • Inventory Items
  • Employees
  • Fixed Assets
  • Intangible Assets

Batch operations are available whenever you see these three buttons in the bottom-right corner of the Manager window:

Batch Create

To begin, click Batch Create. Follow the instructions sequentially. First, copy the blank template from the template box and paste it into your spreadsheet program:

Batch Delete

Batch Delete removes all entries in a tab that have no records associated with them. As with editing throughout Manager, the program will not allow you to delete anything that has already been referenced. Still, Batch Delete is a brute-force tool; it cannot be used to selectively remove entries.

When Batch Delete is clicked, Manager displays a list of all data that will be deleted, showing red strike-through lines to emphasize what will be removed. If you want to proceed, click the red button:

Warning
The Batch Delete operation cannot be reversed. Do not use it unless you are certain you want to permanently remove all data shown on the review screen. And always heed the caution to back up your data before any batch operations.
