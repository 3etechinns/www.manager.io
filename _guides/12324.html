Beginning with version 17.10.46, new installations of Managerâ€™s desktop edition require an activation key. The purpose of this key is to identify the download source. The key is:

  • Free. The key is displayed on the Download page.
  • Anonymous. No personal information is necessary or requested.
  • Independent. No internet connection is required after the program is downloaded. Activation occurs offline in the program.

Note
This Guide does not apply to the server or cloud editions.

Initial activation

The first time the program is launched, an activation screen will appear:

Enter the activation key from the Download page and click Activate. This step confirms your copy of Manager was downloaded from the official web site. A confirmation notice will appear:

Activation is complete. Click Continue to use the program.

Updating

Activation is a one-time event. When an existing installation of Manager is updated, no activation key is required.
