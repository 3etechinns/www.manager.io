Popular questions
Some of the more common questions we get.

If I run Manager in the cloud, can I use my iPad, iPhone or Android?

Yes. Unlike our desktop edition which only runs on Windows, Mac and Ubuntu. The cloud edition enables access to Manager with any web-browser like other online accounting systems.

Is cloud edition compatible with desktop edition?

Yes. We maintain 100% compatibility between the free desktop edition and the cloud edition. You can transfer data between the cloud and desktop edition with ease.

How can I transfer my data from desktop to cloud?

Transferring data between editions is very easy. First you make backup in desktop edition, then import your backup file in cloud edition.

How long is the free trial?

Free trial is for 7 days. After that, you will need to enter your credit card details if you want to continue using the service.

If I'm accountant or bookkeeper, can I create all my clients?

Absolutely. You can create unlimited businesses and unlimited users. When you are giving access to your clients, make sure to set them up as restricted users so they have access to their businesses only.
